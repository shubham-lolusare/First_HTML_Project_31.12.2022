HTML List

Unordered Lists

In Unordered list of the HTML, we have to use the <ul> element, so that we can get the lists with bullets or dots.

For Example, the Unordered list of the names are

  • Shubham
  • Rakhi
  • Rushabh Rao
  • Ranjana Tai
  • Rakesh Rao

Applications of Unordered Lists

We can use this type of List when we want to show the items present in a particular data.

Ordered Lists

In Ordered Lists of the HTML, we have to use the <ol> element, so that we can get the numbered lists

For Example, the Ordered list of the Numbers are

  1. One
  2. Two
  3. Three
  4. Four
  5. Five

Applications of Ordered List

We can use this type of list if we want convey the information where the order is important like in recepies.
